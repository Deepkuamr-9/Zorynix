# 🌟 Overview

Zorynix is an AI-powered mental wellness companion designed to provide a calm, supportive, and judgment-free environment where users can share thoughts, emotions, concerns, and daily challenges.

The platform encourages emotional expression through empathetic conversations while offering practical guidance, wellness-focused interactions, and crisis-awareness resources.

Whether someone is experiencing stress, anxiety, academic pressure, burnout, or simply needs a safe space to talk, Zorynix aims to provide meaningful support through compassionate AI-driven conversations.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/zorynix.git

cd zorynix
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

---

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python -m streamlit run app.py
```

---

# ⚙️ Configuration

Create:

```text
.streamlit/secrets.toml
```

Add:

```toml
ZORYNIX_MODE = "local"
DEFAULT_REGION = "india"
```
